## Installation and Usage

#### npm
``` shell
npm install oreum-ui --save
```
#### yarn
``` shell
yarn add oreum-ui
```

#### Use component `"*.vue"`
```vue
<template>
  <o-button />
</template>

<script setup lang="ts">
import { OButton } from 'oreum-ui'
</script>
```

#### Use styles `"*.vue"` `"*.ts"`
```typescript
import 'oreum-ui/dist/oreum-ui.css'
```
